### Encoding
#### Base64
```bash
Base64 is not encryption, it is an encoding method that converts binary data into readable text using A–Z, a–z, 0–9, +, /.

# encodding text to base64
echo "hello" | base64

# Decoding Base64 to text
echo "aGVsbG8=" | base64 -d
echo "aGVsbG8=" | base64 --decode

base64 Sniff.txt > encoded.txt
base64 -d Sniff.txt > secret.txt
```
