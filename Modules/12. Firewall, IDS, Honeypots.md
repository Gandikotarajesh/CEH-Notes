```bahs
Firewall is a SW/HW used monitor and controlling the  incoming and outgoing traffic based on the rules set.
```
#### Types of Firewalls
```bash
1. Packet filter firewall (Layer 3 - Network Layer)
    1. Traffic is filtered based on IP address, Port numbers and Protocols.
2. Circuit-level gateways (Session Layer)
    1. These FW monitor TCP handshake to determine the requested sessions are legitimate or not.
3. Application Firewall
    1. Used to protect webservers (WAF)
    2. These firewall Examine the traffic and filter on application specific such as HTTP, POST, GET. This works on TCP/IP Model.
4. Stateful firewall (Network Layer)
    1. This combines the aspects of the above three types of FW.
    2. Maintains a table of connections and their status.
```
#### Defence in depth and breath
```bash
Defense in depth involves implementing multiple layers of security controls throughout an IT system. If one layer fails, another layer provides backup protection. This strategy addresses different potential threats at various levels, such as physical, network, application, and data security.

Defense in breadth refers to using a diverse set of security measures and technologies to protect against a wide range of threats and vulnerabilities. It focuses on employing different types of security tools, like firewalls, antivirus software, and intrusion detection systems, to cover different angles of security.

In short, defense in depth is about multiple layers of security, while defense in breadth is about diversity in security measures. Both aim to provide robust protection, but from different perspectives.

One way of addressing defence in depth and defence in breath is to introduce a unified threat management device.
```
