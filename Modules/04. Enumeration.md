#### Enumerating Linux machines using enum4linux
```bash
To perform user enumeration on the target machine using -U option.
enum4linux -U <TargetIPaddress> 

to extract file shares related information
nmap -S <TargetIPaddress>

To extract password policy information
enum4linux -P <target ip>
```

#### Building wordlist using CEWL 
```bash
cewl -d 1 -m 8 <url> -w <outputfilename> 

-d -> Crawl the website to a depth of 1.
-m 8-> Only include words that are 8 or more characters long.
```
