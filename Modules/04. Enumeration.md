
#### SMB Enumeration
```bash
nmap -p 139,445 --open -sV <ip>

Enumerate SMB Shares: 
smbclient -L \\10.10.55.X 

Brute-force SMB Credentials: 
hydra -L user_list.txt -P password_list.txt <ip> smb

Access SMB Share: 
smbclient \\\\10.10.55.X\\share_name -U user%password123

Retrieve and Read file.txt: 
get file.txt
cat file.txt
```
