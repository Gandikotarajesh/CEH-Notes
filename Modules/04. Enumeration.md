
#### SMB Enumeration
```bash
nmap -p 139,445 --open -sV <ip>

Anonymous Login
smbclient -L //10.10.55.50 -N
get access to share with anonymous login
smbclient //10.48.183.82/Anonymous -N
get file.txt

Lists all SMB shares on the server 
smbclient -L //10.10.55.50 -U username
 
Connect to specific share
smbclient //10.10.55.50/Public -U username
ls
->Connects directly to the Public share.
-> Once connected, you'll get the SMB prompt ( smb: \> )

Brute-force SMB Credentials: 
hydra -L user_list.txt -P password_list.txt <ip> smb


Retrieve and Read file.txt: 
get file.txt
cat file.txt
```
