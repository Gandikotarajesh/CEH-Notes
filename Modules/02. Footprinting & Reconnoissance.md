
#### FQDN
```bash
Nmap -p389 -sV <target_list>
389 -> LDAP
FQDN = Host.Domain
FQDN = DC.pentester.com

or
nmap -T4 -A -p 389,636,3268,3269 192.168.x.x/2x
or
nmap -p 389 --script ldap-rootdse <target_IP>
or
nmap -Pn -A <target_IP>

Tip : • Open port 88 + 389 + 445 together → High chance it's a Domain Controller.
```
#### WampServer
```bash
nmap -A -T4 <target-ip>
or
nmap -sV -sC -O <target-ip>
or
nmap -T4 -A -p 80,443 192.168.x.x/2x
nmap -sV -A -p 80 ip/24
```
#### SMB port server checking
```bash
nmap -p 139,445 ip/24
nmap -T4 -sS -p 139, 445 --script vuln ip/24
```
