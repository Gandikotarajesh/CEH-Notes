#### FQDN
```bash
nmap -A -p 445 192.168.0.109 -T4

Nmap -p389 -sV <target_list>
389 -> LDAP
FQDN = Host.Domain
FQDN = DC.pentester.com

or
nmap -T4 -A -p 389,636,3268,3269 192.168.x.x/2x
or
nmap -p 389 --script ldap-rootdse <target_IP>
or
nmap -Pn -A <target_IP>

Tip : • Open port 88 + 389 + 445 together → High chance it's a Domain Controller.
```
#### Find Domain Controller
```bash
Check for 88/TCP kerberos-sec and port 389/TCP LDAP opened which confirms that it is DC
Once Target is Found, Perform the Aggressive scan to get full details about the DC
```
#### Domain controller IP
```bash
Assume that 10.10.1.22 is a machine
nmap shows that host 10.10.1.22 has port 88/TCP kerberos-sec and port 389/TCP LDAP opened which confirms that our DC IP address is 10.10.1.22.
And to find the FQDN (Fully Qualified Domain Name)
`nmap -A -T5 10.10.1.22` → Gives Product version, FQDN and many other.
```

####  Extracting Emails and subdomains details using the harvester
```bash
This tool is to gather emails, subdomains, hosts, employee names, open ports and banners from different public sources like Google, Bing and other search engines.
# Syntax
theHarvester -d <domain> -b <source>
# Search using Google
theHarvester -d example.com -b google
# Search using all available sources
theHarvester -d example.com -b all
# Limit no. of results
theHarvester -d example.com -b bing -l 200
```
