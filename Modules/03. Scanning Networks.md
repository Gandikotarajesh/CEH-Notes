#### Scanning
```bash
nmap -sC -sV -p- -T4 <IP>
```
#### Seveiry scoe or EOL vulnerability
```bash
nmap --script vuln <ip>
or
openvas
```
#### Live Host discovery with fping
```bash
fping -aqg 192.168.0.1/24 
```
#### Live host discovery with netdiscover
```bash
sudo netdiscover -r 192.168.0.1/24 
```
#### Live host discovery with arp-scan
```bash
sudo arp-scan 192.168.0.1/24 
```

#### NMAP Commands
```bash
# Nmap scan if firewall/IDS is opened
nmap -sS -v 10.10.10.16

```
#### NMAP
```bash
# TCP Connect
nmap -sT <ip>

# Syn scan
nmap =sS <ip>

# Version detection
nmap -sV <ip>

# OS Detection
nmap -O <ip>

# UDP scan
nmap -sU <ip>

# Aggressive scan
nmap -A <ip>

```
