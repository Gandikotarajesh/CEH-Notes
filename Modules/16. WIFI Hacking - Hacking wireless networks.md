#### Important terms
```bash
1. Access Point:  Extends a wired network into a wireless one.  Acts as a central transmitter/receiver for Wi-Fi signal, connects to a router or switch via Ethernet.
2. SSID (Service Set Identifier): Identifier used to identify the access point.
3. BSSID (Base Service Set Identifier) : MAC address of the Access Point
4. Bandwidth: Amount of info that can be transferred over the connection.
```

### Analyze the Capture File
```bash
Aircrack-ng is a tool used for breaking WEP and WPA/WPA2-PSK
```
Identify the target network
```bash
use airodump-ng to list the wireless networks captured in the given file airodump-ng <wifi.cap>
Note down the BSSID
```
Capture Traffic for the Target Network 
```bash
airodump-ng --bssid BSSID --channel CHANNEL -w outputfile W!F!_Pcap.cap 
```
Crack the Wi-Fi Password
```bash
Use aircrack-ng with the captured data to attempt to crack the Wi-Fi password.
aircrack-ng -w /path/to/wordlist.txt outputfile-01.cap 
```
