#### Important terms
```bash
1. Access Point:  Extends a wired network into a wireless one.  Acts as a central transmitter/receiver for Wi-Fi signal, connects to a router or switch via Ethernet.
2. SSID (Service Set Identifier): Identifier used to identify the access point.
3. BSSID (Base Service Set Identifier) : MAC address of the Access Point
4. Bandwidth: Amount of info that can be transferred over the connection.
```
#### Quick Commands
```bash
- Cracking `.cap` → `aircrack-ng -w wordlist.txt capture.cap` or `aircrack-ng -b <bssid> -w wifiPassList.txt handShakeCapture.cap`
    - To fine the `bssid` given the .cap file `aircrack-ng <filename>.cap`
```
#### WPA/WPA2 Password Cracking (Given .cap file)
```bash
aircrack-ng WPA2crack-01.cap
here we will get BSSID.

# crack password
aircrack-ng -w password.txt WPA2crack-01.cap

# if there is multple ips
aircrack-ng -a2 -b 22:7F:AC:6D:E6:8B -w password.txt WPA2crack-01.cap

-a1 --> WEP attack
-a2 -> WPA/WPA2-PSK attack
```
### WPA/WPA2 Password Cracking (Need to Capture Handshake)
```bash
# checking the available networks .
ip a

# to get list of devices
iwconfig

# Enable monitor mode
airmon-ng start wlan0

# to see all the available networks
airodump-ng <NetworkInterfaceName> 
airodump-ng wlan0mon

# Capture handshake
airodump-ng --bssid <BSSID> --channel <CH> -w capture wlan0mon

# Crack password
aircrack-ng -a1/a2 --bssid -w password.txt capture-01.cap

-a1 --> WEP attack
-a2 -> WPA/WPA2-PSK attack
```
