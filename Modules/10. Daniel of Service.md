#### Types of DDOD attacks
```bash
1. ICMP Flooding → many ping requests to the target. When the attacker send ‘n’ ping requests target should reply with ‘n’ responses.
2. SYN attack → Uses TCP (way handshake), attacker is only going to send SYN request where as target replies with ACK + SYN in this case the attacker is not going to reply to the target response.
3. UDP Flooding → Sends lakhs of UDP packets to the target. As target system becomes overwhelmed and unresponsive to other clients.
4. Application Based DoS attack → Attackers target vulnerabilities in specific applications or services like SMTP etc.
5. HTTP → Targeting only HTTP services. [GET or POST] requests.
```
